Essence of Progress Bars

A progress bar tells how much of a computation is completed. It can be updated periodically or it can count the size of processed chunks. To update your progress bar, you must either have the current total progress or the amount to increase since the last update.

tqdm is a progress bar package for Python. Here is a nice gif showing several features of tqdm: link to gif

Below is an example which you can find on the main page of tqdm. Here we are giving the update amount (using pbar.update) to update the progress bar:

with tqdm(total=100) as pbar:
    for i in range(10):
        sleep(0.1)
        pbar.update(10)

Giving the current total progress would look something like this: (This example doesn't run because tqdm doesn't give an option to use total progress in update. The example is given only for illustrative purposes.)

with tqdm(total=100) as pbar:
        current = 0
    for i in range(10):
        sleep(0.1)
        current += 10
        pbar.update(current_total=current)
